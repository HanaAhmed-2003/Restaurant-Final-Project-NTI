# Restaurant-Final-Project-NTI
A full end-to-end data analysis project covering data cleaning, SQL querying, Excel pivot reporting, and Power BI dashboarding for a multi-branch international restaurant chain.

## Project Overview

This project was submitted as the **final project for the Data Analysis track at the National Telecommunications Institute (NTI)**. It simulates a real-world business scenario where a restaurant chain operating across **5 European branches** needs insights into sales performance, customer behavior, loyalty programs, and ordering patterns.

The analysis covers the complete data pipeline from raw, dirty data to clean, structured insights presented through interactive dashboards and pivot reports.

## Dataset Description

The dataset consists of **4 relational tables**:

customers: Customer demographics, city, registration date, loyalty status
branches: Branch ID, city, country, manager name
menu: Item ID, item name, category, unit price
orders: Order transactions linking customers, branches, and menu items

## Tools Used
Python(Pandas, NumPy)
SQL (SQL Server)
Excel Pivot tables
Power BI
